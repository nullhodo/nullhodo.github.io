## Development

This project uses Astro.

### Setup

```bash
pnpm install
```

### Commands

| Command            | Action                                 |
| :----------------- | :------------------------------------- |
| `pnpm dev`         | Starts local dev server                |
| `pnpm build`       | Build the production site to `./dist/` |
| `pnpm preview`     | Preview the build locally              |
