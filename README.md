## Development

This project uses Astro.

### Setup

```bash
npm install
```

### Commands

| Command           | Action                                 |
| :---------------- | :------------------------------------- |
| `npm run dev`     | Starts local dev server                |
| `npm run build`   | Build the production site to `./dist/` |
| `npm run preview` | Preview the build locally              |
